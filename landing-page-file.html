PKC Landing Page — Local File Mode

This variant is designed to run directly from file:// without any external CDNs or server-only APIs. It showcases the GovTech/PKC principle that rich documentation and P2P experiments can be driven by static assets and locally provisioned JavaScript.

Mode: file protocol Static-first · Local-first

What Works Here

Open this page via a path like: file:///Users/<you>/Documents/Development/GovTech/PKC/LandingPage/landing-page-file.html and you can:

  • Browse PKC markdown docs rendered by local scripts.
  • Inspect P2P / WebRTC demos as static HTML+JS bundles.
  • Verify that no external CSS/JS CDNs are required.

The more network-heavy dashboards (e.g. those requiring WebSocket servers) are still available, but may need http:// hosting for full behavior. This page itself never calls /api endpoints or remote origins.
